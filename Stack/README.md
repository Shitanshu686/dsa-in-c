# Stack

## Language
C

## Description
This program implements the Stack data structure using the C programming language.

## Operations Implemented

- Push
- Pop
- Peek (Top)
- Display

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Push      | O(1)       |
| Pop       | O(1)       |
| Peek      | O(1)       |
| Display   | O(n)       |

## How to Run

1. Compile the program:
   ```bash
   gcc stack.c -o stack
   ```

2. Run the executable:
   ```bash
   ./stack
   ```
