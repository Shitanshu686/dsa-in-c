# Binary Search Tree (BST) in C

This repository contains the implementation of the **Binary Search Tree (BST)** data structure in C.

## Features
- Create a Binary Search Tree
- Insert a node
- Search for a node
- Delete a node
- Inorder Traversal
- Preorder Traversal
- Postorder Traversal

## Time Complexity

| Operation | Average | Worst |
|-----------|---------|-------|
| Search | O(log n) | O(n) |
| Insert | O(log n) | O(n) |
| Delete | O(log n) | O(n) |

## Concepts Covered
- Binary Search Tree
- Recursion
- Dynamic Memory Allocation
- Tree Traversals
- Pointer Manipulation

## Technologies Used
- C Programming
- GCC Compiler

## How to Run

```bash
gcc bst.c -o bst
./bst
```

## Learning Outcome
This project demonstrates the implementation and working of the Binary Search Tree (BST), including insertion, deletion, searching, and traversal operations.
